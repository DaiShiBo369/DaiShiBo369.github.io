# 个人音乐馆 — 构建计划

> 创建日期：2026-04-29

## 项目定位

独立于 Hexo 博客的个人音乐馆网页。展示喜欢的专辑封面，写出与音乐的故事，嵌入 QQ 音乐播放器在线收听。

## 技术方案

- **单 HTML 文件**，所有 CSS 和 JS 内联
- 双击即可在浏览器中打开，零依赖零构建
- 专辑数据以 JSON 数组形式写在文件底部，修改数据即可增删专辑
- 响应式布局，适配手机和桌面

## 文件位置

`D:/My_Obsidian_Vault/Blog/music-gallery.html`

## UI 设计 — "云图"风格

### 视觉概念
以天空、云层、光影为视觉语言，营造漂浮、沉浸的音乐氛围。

### 背景效果
- 多段天空渐变，从深蓝夜空过渡到天蓝、粉紫日落
- CSS 绘制的多层云朵，不同速度缓慢漂浮
- 滚动时背景色微妙变化

### 专辑卡片
- 毛玻璃效果（backdrop-filter: blur）
- 专辑封面 + 歌手名 + 年份
- 点击展开个人故事
- 内嵌 QQ 音乐 iframe 迷你播放器
- "在 QQ 音乐中打开" 外链按钮

### 配色方案
```
天空渐变：#0a1628 → #1a3a5c → #4a90d9 → #87ceeb → #f0e6ff
云朵：rgba(255,255,255,0.3) ~ rgba(255,255,255,0.8)
卡片：rgba(255,255,255,0.15) + 毛玻璃
强调色：#ff6b9d（日落粉）
文字：#ffffff（主）/ rgba(255,255,255,0.7)（次）
```

### 字体
- Quicksand（标题，Google Fonts CDN）
- 系统无衬线字体（正文）

## QQ 音乐嵌入方式

使用官方 iframe 播放器：
```html
<iframe
  frameborder="no" border="0"
  marginwidth="0" marginheight="0"
  width="100%" height="80"
  src="https://i.y.qq.com/v8/playsong.html?songmid=SONG_MID">
</iframe>
```

获取 songmid：在 QQ 音乐网页版打开歌曲详情页，URL 中 `songDetail/` 后面的字符串即为 songmid。

## 数据结构

```js
const albums = [
  {
    title: "专辑名",
    artist: "歌手名",
    cover: "封面图URL",
    year: "2024",
    story: "我和这张专辑的故事...",
    songs: [
      { name: "歌曲名", songmid: "003aAYrm3GE0Xg" }
    ],
    tags: ["标签1", "标签2"]
  }
];
```

## 页面结构

1. **Hero 区域**：大标题 + 副标题 + 漂浮云朵 + 滚动提示
2. **专辑画廊**：响应式网格，卡片展示所有专辑
3. **底部**：版权信息 + 返回顶部

## 交互功能

- 悬停卡片上浮 + 阴影加深
- 点击展开/收起故事全文
- QQ 音乐播放器嵌入播放
- 滚动视差效果
- 返回顶部按钮

## 使用方式

1. 双击 `music-gallery.html` 在浏览器中打开
2. 修改文件底部的 `albums` 数组添加自己的专辑
3. 封面图片可使用图床 URL 或本地相对路径
4. 每首歌的 `songmid` 从 QQ 音乐网页版获取
